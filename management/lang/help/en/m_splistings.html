The following section is intended for management of Sponsored Links. When a search is done using Sponsored Link keyword the link will be shown in an highlighted mode at the page top. When this section is entered you are displayed the following information:
Keyword This is the keyword with which your site will bee associate. Keyword can contain only Latin letters and the number, the remaining symbols are not allowed.
Bid This is the price which you give to the keyword. After people clicks into your url this price will be substract from your balance. Remember than higher bid than closer your link to the top of page. Bid Listing must be greater then the minimum, which set Administrator.
Title Title under which your link to the site will be appear. Title will be displayed in Search results page.
Registered The date of keyword registration.
Status If listing status is "Approve"' the description will be displayed in Search results. If listing status is "Pending"' the description will be displayed in Search results after Administrator approve.
Rank It represent the actual rank of member site in Sponsored links campaign.
Note: This field can be disabled from the Administator.
Bid for top The system compares all the keyword listing and compare the bids made by members for the same keyword and calculate the highest bid. You can change the bid to appear in the first position using up arrow button.
