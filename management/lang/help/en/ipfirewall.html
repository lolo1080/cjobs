IP FireWall Use this feature you can deny access to your site from specifically ip addresses. Set "Security Level" for locked ip addresses. Also you can deny adresses by mask. Use asterisk ("*") in the one of the fields when enter IP.
For example: if you enter 198.123.45.* all the addresses which begins from 198.123.45 will be locked.
Use this feture carefully. Click on "Common settings" menu item and choose a "Security Level" for locked ip addresses.
