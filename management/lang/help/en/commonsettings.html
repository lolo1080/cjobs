Security Level Security Level option enables Administrator to set constrains on searches and clicks being originated from banned countries or IP addresses.
Low - allows searches and clicks from all countries, from any IP address. This level is the lowest one.
Medium - rejects clicks from banned IP address and countries. This is a medium level of security. It allows searches and clicks from banned countries and IP addresses, but only in case they are without earnings. If Affiliates execute click or search from banned country or IP address, then they are not going to be paid.
High - rejects searches from banned IP addresses and countries - This is high level of security. Any attempt to execute a search, click, or sign in for an account will result in block of such actions with displaying the message Your country is blocked or Your IP [IP address] is blocked. We recommend this level to protect you from any kind of fraud.
Default Pay for search,
Pay for search interval		 Default Pay for Search it is the payment for each search executed by member. Pay for search interval means that the member will be paid for each seach over interval in hours. "All Times" means that member will be paid for each search executed by him. Default Pay for Search settings are global and are applicable to all Affiliates. Administrator is also enabled to set individual settings for any Affiliate using the Pay for search option in Members Settings > Edit this Affiliate configuration.
To see Search Statictics (earned/not earned money).
Click "Users"at the top of page, then choose user. Click "Search Keyword Statistics (earned money)" if you want to see what earned user for searches. Or click "Search Keyword Statistics ( not earned money)" if you want to see all of the searches which hi did.
REMEMBER This Settings are global for all of users but you can personalize pay for searches for any user.
Click "Users" then choose user. Click "Edit this affiliate configuration" than check "Use this affiliate configuration by default for user" and set payment for searches and time interval after which you will pay to user personally.
NOTE If you check "Use this affiliate configuration for .. by default" all of settings in affiliate configuration will be in use by default not ONLY Pay for Search.
Alternative search The PPC system has three different types of built-in alternative search engine such Altavista, MSN, and Google. Administrator can easily select which of the alternative search engines to use in the PPC system. Also he can disable this option and do not use any alternative search engine at all. If you select the option Disable, then no alternative links will be found in the search results. The actual search engine will be written in blue characters.
Amount of links The default amount of alternative links to show on each search results page.
Price Price is the field to fill in the prize of one Sponsor box during his display in Search results.
