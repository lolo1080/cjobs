PayPal Settings If you are using PayPal payment system, set "Accept PayPal payment" parameter to "Yes".
Your PayPal e-mail - your PayPal e-mail.
Minimum withdraw by PayPal - minimum amount a user can withdraw from site by PayPal payment system.
Minimum deposit by PayPal - minimum amount a user can deposit by PayPal payment system.
E-Gold Settings If you are using E-Gold payment system, set "Accept E-Gold payment" parameter to "Yes".
Your E-Gold id number - your E-Gold id number.
Your E-Gold passphrase - your E-Gold passphrase.
Minimum withdraw by E-Ggold - minimum amount a user can withdraw from site by E-Ggold payment system.
Minimum deposit by E-Ggold - minimum amount a user can deposit by E-Ggold payment system.
Cickbank Settings If you are using Cickbank to accept credit cards, set "Cickbank credit card payment" parameter to "Yes".
Your Clickbank id number - your Clickbank id number.
Minimum withdraw by Clickbank - minimum amount a user can withdraw from site by Clickbank credit card payment system.
Minimum deposit by Clickbank - minimum amount a user can deposit by Clickbank credit card payment system.
Clickbank URLs - the Clickbank URLs. Enter the amount of money, followed by the appropriate Clickbank URL, with a ";" symbol in between. Seperate each with a "Enter"
IMPORTANT: You will also need to change the return URL to the location of ccpay_clickbank.php on your site. Login to your account at Clickbank, click on "_________" then "__________", then enter the URL and save.
2checkout Settings If you are using 2checkout to accept credit cards, set "2checkout credit card payment" parameter to "Yes".
Your 2checkout id number - your 2checkout id number.
Minimum withdraw by 2checkout - minimum amount a user can withdraw from site by 2checkout credit card payment system.
Minimum deposit by 2checkout - minimum amount a user can deposit by 2checkout credit card payment system.
2checkout URL - the 2checkout URL.
2checkout payment process in test mode - set this to "Yes" if you want to put the 2checkout payment process in test mode.
IMPORTANT: You will also need to change the return URL to the location of ccpay_2checkout.php on your site. Login to your account at 2checkout, click on "Shopping Cart" then "Cart Details", then enter the URL and save.
